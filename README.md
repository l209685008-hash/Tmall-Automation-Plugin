# 天猫自动上架与 SKU 图片填充 Chrome 插件

这是可安装到 `chrome://extensions` 的 MV3 Chrome 扩展，已合并“天猫自动上架”和“SKU 图片填充”两套能力，并使用 Chrome 右侧面板作为主操作界面。

## 安装

1. 打开 Chrome：`chrome://extensions`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择此文件夹：`C:\Users\Administrator\Desktop\天猫项目\天猫自动上架Chrome插件`
5. 如果需要读取默认表 `大箱价格.xlsx`，在扩展详情里开启“允许访问文件网址”

需要 Chrome 114 或更高版本。点击扩展图标会直接打开右侧面板；切换商品标签页时，面板会自动刷新当前页面状态。

扩展会请求 `debugger` 权限，仅用于在当前天猫发布页模拟真实鼠标点击打开 Fusion 下拉框；不会调用隐藏接口，也不会点击最终发布/保存按钮。

## 自动上架

1. 在 Chrome 打开并登录天猫/淘宝商品发布页。
2. 点击扩展图标打开右侧面板。
3. 导入 `大箱价格.xlsx`，或点击“读取默认表”。
4. 依次运行“诊断页面”“填写文本信息”“选择下拉属性”“上架时间：立刻上架”“导入/填充 SKU”“SKU助手：价格取整”，或直接点“全流程准备”。

插件会阻止最终类按钮：`提交`、`发布`、`保存`、`保存草稿`、`放入仓库`、`立即上架`、`确认发布`、`保存并发布`。主图仍可人工上传。

## SKU 图片填充

1. 在发布页的“销售信息”里确认 SKU 已经生成。
2. 如果还没有打开 SKU 图片弹窗，点扩展里的“SKU 图片填充 / 打开填充弹窗”。
3. 在弹窗左侧的网页素材库进入“小箱sku”素材文件夹。
4. 点“扫描网页素材”确认识别到右侧 SKU 和左侧网页素材图。
5. 点“按网页图片名尺寸填充”。例如 `30-15.jpg` 会匹配 `【优质三防横版】30*15*2000张*120卷/箱`。
6. 插件会逐行等待缩略图真实出现；点击未落地时自动重试并清理残留选中状态。填充结束后保留弹窗，不自动点击右下角“确定”，并在右侧面板报告漏填数量与具体 SKU。

图片可以重复使用：同一张 `30-15.jpg` 可以填入多个包含 `30*15` 的 SKU。插件只点击网页素材库中已有图片，不读取本地图片文件夹，也不走本地上传。

同一文件名在素材库中重复上传时，会按“文件名 + 商品尺寸 + 颜色”折叠为一个逻辑素材，不再因不同 CDN 地址产生错误冲突。多个 SKU 需要同一张图时，插件会逐个 SKU 复用该素材，并分别验证缩略图是否落地。

为避免左右列表顺序不同导致尺寸错配，“按当前可见顺序填充”保持禁用。

## 固定字段

- 导购标题：打印清晰粘性好整箱包装性价比高
- 商品标题：禹尚热敏标签纸打印纸整箱装
- 商品卖点：打印清晰，整箱批发，工厂直销粘性强不卡纸
- 品牌：禹尚
- 材质：热敏纸
- 生产企业：义乌市腾望纸业有限公司
- 适用品牌：GODEx、启锐、快麦、HPRT/汉印、佳博、ZEBRA/斑马、科诚、GPRINTER、GODEX、Deli/得力、brother/兄弟、Argox/立象科技
- 材质特性：打印清晰、粘性强、防水、防油、防刮、三防
- 纸张版型：方型
- 上架时间：立刻上架

## 分发

可分发压缩包：`C:\Users\Administrator\Desktop\天猫项目\天猫自动上架Chrome插件.zip`

对方电脑解压后，在 `chrome://extensions` 用“加载已解压的扩展程序”选择解压后的文件夹。
